# Cómo crear un repositorio en GitHub

Sigue estos pasos para crear el repositorio donde subiremos tu web:

1.  **Inicia sesión** en [GitHub.com](https://github.com/login).
2.  En la esquina superior derecha, haz clic en el icono **+** (junto a tu foto de perfil) y selecciona **"New repository"** (Nuevo repositorio).
3.  En la página "Create a new repository":
    *   **Repository name**: Escribe un nombre, por ejemplo: `estilsofa-nextjs`
    *   **Description** (Opcional): "Web de EstilSofa en Next.js"
    *   **Public/Private**: Selecciona **Private** (Privado) si no quieres que el código sea visible para todo el mundo, o **Public** si prefieres código abierto.
    *   **Initialize this repository with**: **¡NO MARQUES NADA!** (Déjalo todo desmarcado: ni README, ni .gitignore, ni License). Esto es muy importante para que podamos subir el código que ya tenemos.
4.  Haz clic en el botón verde **"Create repository"**.
5.  Una vez creado, verás una pantalla con instrucciones. Busca la sección que dice **"…or push an existing repository from the command line"**.
6.  Copia la URL que aparece ahí. Debería ser algo como:
    *   HTTPS: `https://github.com/TU-USUARIO/estilsofa-nextjs.git`
    *   SSH: `git@github.com:TU-USUARIO/estilsofa-nextjs.git`

**¡Pásame esa URL y yo me encargo del resto!**
